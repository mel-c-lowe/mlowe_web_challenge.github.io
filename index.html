Weather Analysis

The purpose of this project was to analyze weather changes in conjunction with latitude across the globe. To gather data, we used the OpenWeatherMap API to collect data from approximately 500 cities.

After the dataset was assembled and cleaned, we used MatPlotlib to plot various aspects of the data collected, such as temperature, humidity, cloudiness, and wind speed, all compared to the latitude of the given city. This site provides the visualizations generated through that analysis with notes regarding trends, or lack thereof, in the data.
